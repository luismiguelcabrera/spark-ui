# Spark UI — Improvement Plan

## Status Legend
- [ ] Not started
- [~] In progress
- [x] Completed

## Completed (Wave 1)
- [x] **Alert** (feedback) — Standalone inline alert box with icon, title, description, dismiss. 4 variants, dismissible, a11y tested.
- [x] **AutoResize Textarea** (forms) — `autoResize`, `minRows`, `maxRows` props. Grows with content, respects max height.
- [x] **InputGroup / InputAddon** (forms) — InputGroup, InputLeftAddon, InputRightAddon, InputLeftElement, InputRightElement. 3 sizes, clickable overlays.
- [x] **Vertical Tabs** (navigation) — `orientation` prop on Tabs. Vertical layout with border-right indicator. Fixed legacy a11y (role="tab", aria-selected).
- [x] **MultiSelect** (forms) — Searchable multi-value select with chips, clear-all, keyboard nav, maxSelections, disabled options. 3 sizes.
- [x] **DataTable sorting** (data-display) — Column `sortable`/`sortFn` props, controlled + uncontrolled sort state, asc/desc/none cycle, sort indicator icons. Added ChevronsUpDownIcon.
- [x] **Breadcrumb improvements** (navigation) — Removed Next.js dep, custom separators, per-item icons, collapsible overflow with maxItems, semantic HTML, forwardRef.
- [x] **Image** (data-display) — Lazy loading, skeleton placeholder, error fallback, 6 radius variants, objectFit, aspectRatio, fade-in transition.

## Completed (Wave 2)
- [x] **useAsync hook** (hooks) — idle/loading/success/error states, execute with args, stale response handling, reset. 10 tests.
- [x] **useWindowScroll hook** (hooks) — Track x/y scroll position, scrollTo/scrollToTop/scrollToBottom utilities. 7 tests.
- [x] **useUpdateEffect hook** (hooks) — useEffect that skips initial mount, fires only on dep changes. 6 tests.
- [x] **SortableList / DragHandle** (data-display) — Pointer drag + keyboard reorder (Space/Arrow/Escape), GripVerticalIcon, renderItem pattern. 19 tests.
- [x] **VirtualList** (data-display) — Zero-dep windowed rendering, fixed item height, overscan, getKey, scroll update. 16 tests.
- [x] **Dialog** (feedback) — Declarative + imperative (useDialog) confirm/alert, promise-based API, danger variant, loading. 25 tests.

## Up Next (Wave 3)
- [x] **Tooltip improvements** (feedback) — Arrow indicator, openDelay/closeDelay, controlled open/onOpenChange, disabled, forwardRef. 22 tests.
- [x] **Progress steps** (data-display) — Bar with positioned step markers, check icons, 3 sizes, 4 colors, descriptions. 16 tests.
- [x] **useOnClickOutside hook** (hooks) — Multi-ref support, mousedown+touchstart, enabled flag, stable handler ref. 6 tests.
- [x] **Kbd improvements** (data-display) — combo prop ("Ctrl+K"), platformAware (⌘/⌥/⇧ on Mac), Mod key, auto-capitalize. 7 tests.
- [x] **DataTable filtering** (data-display) — Text input + select dropdown filters, custom filterFn, controlled/uncontrolled, filter→sort pipeline. 11 tests.
- [ ] **Drawer improvements** (feedback) — Swipe-to-close gesture, size variants, placement. Medium complexity.
