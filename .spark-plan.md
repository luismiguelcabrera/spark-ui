## DEDUPLICATION NOTES

Before adding any icon, verify:
1. **Not already in `icons.tsx`** under a different name
2. **Not aliased in `registry.ts`** — some icons are registered under multiple names
3. **Visually distinct** from existing icons — e.g., `Edit` vs `PenSquare` are similar
4. Icons that share semantic meaning but different visuals are OK (e.g., `Trash` and `Trash2`)

### Known Existing Aliases to Avoid Duplication:
| Alias | Maps To |
|-------|---------|
| `x`, `close` | CloseIcon |
| `add`, `plus` | PlusIcon |
| `delete`, `trash` | TrashIcon |
| `error` | AlertCircleIcon |
| `warning` | AlertTriangleIcon |
| `visibility` | EyeIcon |
| `visibility_off` | EyeOffIcon |
| `notifications` | BellIcon |
| `light_mode` | SunIcon |
| `dark_mode` | MoonIcon |
| `cut` | ScissorsIcon |
| `print` | PrinterIcon |
| `mouse-pointer` | CursorIcon |

## For Each New Icon — Checklist:
1. Add SVG component in `src/icons/icons.tsx` under correct category section
2. Register in `src/icons/registry.ts` with kebab-case + Material Symbols aliases
3. Verify SVG uses `currentColor` for stroke/fill
4. Verify 24x24 viewBox with stroke-width 2
5. Add to barrel export in `src/icons/index.ts` if needed
6. No duplicate names or conflicting visual designs
7. Update Storybook icon gallery story
8. SVG paths sourced from Lucide (MIT license) or hand-drawn

## SVG Source Strategy
- **Primary source: Lucide** (MIT license, 24x24, stroke-width 2, currentColor) — matches our existing icon style perfectly
- **Secondary: Heroicons outline** (MIT, 24x24, stroke-width 1.5) — needs stroke-width adjustment
- **Tertiary: Custom/hand-drawn** — for any icons not in open-source libraries
